Why learn?
Programming might just seem like writing words, which it is, but those words can create new things. You can create new things that people around the world can use. Programming is also a very useful skill for the future as many things will be digital, even more than now.
What Will You Learn?

Python is a programming language well suited for beginners. It is used by professionals from all over the globe and has many applications such as Artificial Intelligence.


2. Web Development
You will learn HTML5 and CSS3 and how to use them to create beautifal websites.


3. Dart and Flutter
Dart is a programming language and flutter is a framework that uses dart. You can use flutter to create beautifal and modern apps for mobile devices.
